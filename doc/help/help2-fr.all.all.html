Les préférences du module Annuaires de liens

Lorsque le module Annuaire de liens est déployé au niveau de la colonne gauche de l’applicatif, l’icône s’affiche.

Elle permet de définir les préférences du module, à savoir :

  • « Nombre de liens par page » : Permet de définir le nombre de liens à afficher sur la page de liste des liens du site.
  • « Texte pour dossier vide » : Permet de définir le texte s’affichant si aucun lien n’est disponible sur la page de liste des liens du site.

Une fois l’ensemble des champs remplis :

  • Cliquer sur pour enregistrer les préférences de l'annuaire de liens.
  • Cliquer sur pour enregistrer les préférences de l'annuaire de liens tout en fermant ensuite la fenêtre des préférences.
