Comment créer une thématique afin de classer les liens

Les liens peuvent être classés par thématique.

Pour créer une thématique :

  • Se rendre dans le module Sites et pages > Rubrique Pages externes
  • Cliquer sur dans la colonne gauche
  • Remplir les champs relatifs aux propriétés de la rubrique.
  • Une fois la rubrique créée, revenir dans l'Annuaire de liens.
  • Ouvrir le module Sites et pages dans l'arbre de ressources
  • Insérer la rubrique précédemment créée depuis l'arbre de ressources dans le dossier racine de l'Annuaire de liens (colonne de gauche de l'applicatif).
