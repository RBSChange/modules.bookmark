Création de pages externes

L’Annuaire des liens permet de créer et de gérer la liste des liens proposés sur le site.

Préalablement à la création d’un annuaire de lien, il faut créer les pages qui seront ensuite appelées via l’annuaire de lien :

  • Se rendre dans le module « Sites et Pages »
  • Créer éventuellement des rubriques afin de classer les liens par thématique en cliquant sur « Créer une rubrique »
  • Créer une page externe en cliquant sur l’icône
  • Remplir les propriétés de la page externe au niveau de la fenêtre s’affichant alors :

Les champs obligatoires sont marqués par.

Les champs non-valides sont marqués par l’icône.

  • Remplir les champs relatifs aux critères généraux de la page :
    • « Nom de page » : Champ textuel libre permettant d’indiquer le nom de la page uniquement visible sous Change.
    • «Texte de lien» : Ce titre sera repris dans le menu de navigation s’il est décidé ensuite de rendre la page visible dans le menu de navigation (voir plus bas pour déterminer la visibilité d’une page).
    • « URL de la page » : Champ textuel libre pour indiquer l’adresse URL du lien (attention à respecter la bonne syntaxe : http://www...)
  • Procéder comme pour la création d’une autre page pour les autres onglets des propriétés de la page.

Une fois l’ensemble des propriétés de la page indiquées, possibilité est offerte de cliquer sur :

  • pour enregistrer les préférences de la page.
  • pour enregistrer les préférences de la page tout en fermant ensuite la fenêtre des préférences.

Pour supprimer une page :

    • Cliquer sur son titre au centre de l’applicatif pour le sélectionner
